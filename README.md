# NOAEM Cleaning Services - Professional Mobile Application

Advanced workforce management system for cleaning and maintenance services with real-time attendance tracking, professional branding, and Supabase integration.

## Key Features

- **Professional Logo & Branding**  
  - Programmatically generated NOAEM logo with teal circular frame, navy bucket, and green broom
  - Brand-consistent theme with dark navy, bright teal, and forest green
  - Responsive icon component for app badge

- **Network Connectivity**  
  - Real-time internet connectivity detection
  - Graceful fallback to local caching when offline
  - Toast notifications for network status

- **Attendance Management**  
  - Supervisor shift tracking
  - Employee attendance recording
  - Location-based assignments
  - Real-time CEO notifications

- **Multi-Role Dashboard**  
  - CEO: Institution & payroll management
  - Supervisor: Team attendance submission
  - Employee: Shift management
  - Admin: System configuration

## Tech Stack

- **Frontend**: Kotlin + Jetpack Compose
- **Backend**: Supabase (PostgreSQL + Realtime)
- **Local Storage**: Room Database
- **Build**: Android Gradle 36
